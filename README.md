# RISC-V-esq processor made in[ Digital by Hneemann](https://github.com/hneemann/Digital)
A homemade CPU architecutre that follows the ideas of [RISC-V](https://en.wikipedia.org/wiki/RISC-V).

Much more will be added here in a more feature complete CPU. Currently (I believe), the 8x version is [Turing Complete](https://en.wikipedia.org/wiki/Turing_completeness). Probably still a load of bugs though.
