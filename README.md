# RISC-V-esq processor made it Digital by Hneemann
A homemade cpu architecutre that follows the ideas of [RISC-V](https://en.wikipedia.org/wiki/RISC-V).

Much more will be added here in a more feature complete CPU. Currently (I beleive) it is [Turing Complete](https://en.wikipedia.org/wiki/Turing_completeness).
